无处不在的抉择之选择控制结构

发表于 2022-10-25 更新于 2022-10-26 分类于 Programmming 本文字数： 3.1k 阅读时长 ≈ 3 分钟
关于C语言选择控制结构的理解

引入

实际问题中需要根据情况的不同选择相应的处理方法,例如:

  • 购物的满减优惠活动

  • 根据收入水平进行纳税

  • 根据判别式求一元二次方程的根

类似上述需要进行”分类讨论”的情形,即为选择控制结构所适用的场景.
对这些需分情况处理的问题进行抽象,首先要描述判断条件(condition),然后使用分支选择结构根据判断条件对不同的情况进行相应处理.

判断条件的表达

  • C语言使用 表达式(Expression) 对判断条件进行描述.
    具体而言:

    • 对于简单的条件,使用关系表达式(Relation Expression)进行描述
    • 对于复杂的条件,使用逻辑表达式(Logical Expression)进行描述

    关系表达式和逻辑表达式之间的关系:

    通过逻辑运算符连接若干个关系表达式或者逻辑条件形成逻辑表达式

    以上两种表达式的取值只有两种:"真"(true)和"假"(false).C语言的基本数据类型中不包括布尔(bool)类型,而是使用非0值表示"真",使用0值表示"假"

  • 关系表达式
    关系表达式由关系运算符和操作数(oprand)组成

    • 关系运算符即比较运算符.
      C语言中的关系运算符：
      C语言中的关系运算符
      符号含义优先级
      < 小于高
      <= 小于等于
      > 大于
      >= 大于等于
      == 等于 低
      !=不等于
    • 操作数包括常量、变量和表达式

  • 逻辑表达式
    逻辑表达式由逻辑运算符和操作数组成

    • 逻辑运算符
      包括"与(and)"、"或(or)"、"非(not)".
      C语言中的逻辑运算符
      符号类型含义 结合性优先级
      ! 单目运算符 逻辑非右结合(从右到左)最高
      && 双目运算符逻辑与左结合(从左到右)较高
      || 双目运算符逻辑或左结合(从左到右)较低
      其中双目运算符"&&"和"||"具有"短路特性": 若表达式的值可由先计算的左操作数单独推导,则不再计算右操作数的值.
      • 运算符&&左侧操作数的值为"假"时,不再计算右侧的操作数的值
        运算符&&左侧操作数的值为"真"时,才计算右侧的操作数的值
      • 运算符||左侧操作数的值为"真"时,不再计算右侧操作数的值
        运算符||左侧操作数的值为"假"时,才计算右侧的操作数的值

      短路特性的应用:

      • 运算符&&左侧的条件可作为前提条件
        例如确保除数不为0
        i != 0 && j % i == 0
    • 操作数包括关系表达式和逻辑变量
      C语言中用整型变量表示逻辑变量.变量值为非0表示"真",0表示"假"

    • if else

      不满足条件condition,则执行statements2

      条件表达式的基本形式为:
      condition ? exp1 : exp2

    • 级联式的if语句:else-if

      if else可以嵌套使用,用以进行深层次的判断

    • switch case

      e.g
      根据运算符op对num1和num2进行运算并输出结果的函数

      switch case语句由if else语句演化而来,其可以更清晰地表达选择分支结构.同时也可与if else嵌套使用
      case后面的值必须为常量,因此具有一点的局限性.
      注意事项：

      • switch语句块中不能定义变量,因为编译器认为这种写法会错变量定义.
      • case语句块执行完后需使用break进行跳出,否则都将归到default的情况中处理
